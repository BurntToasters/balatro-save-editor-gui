Balatro Save Editor GUI

A desktop Balatro save file editor for save.jkr on Windows, macOS, and Linux. Built for fast edits, safety backups, and byte-accurate validation.

Download Latest Release View Source on GitHub

Why This Balatro Save Editor

  • Edit money, chips, hand multipliers, and slot limits.
  • Manage joker metadata and experimental joker operations.
  • Create a timestamped .bak before every write.
  • Validate every write by reload and recompression checks.

Supported Platforms

Installers are available for:

  • Windows x64 (.exe)
  • macOS universal (.dmg)
  • Linux x64 (.AppImage)

Common Save Locations

  • Windows: %APPDATA%/Balatro/<profile>/save.jkr
  • macOS: ~/Library/Application Support/Balatro/<profile>/save.jkr
  • Linux: Steam Proton compatdata/2379780/...

FAQ

Is this a Balatro save.jkr editor GUI?

Yes. It is a desktop GUI for editing Balatro save.jkr files.

Is it safe to use?

Each write creates a backup and then validates the resulting save file.
